# Memory Analyzer Prompt

## Purpose
Extract Master Memory from historical records and biographical material.

## Required Inputs
- `source_materials`
- `chronology_notes`
- `social_relations`

## Output Contract
Return Markdown with sections:
- `Core Values`
- `Timeline Milestones`
- `Geography Path`
- `Political/Intellectual Stance`
- `Citable Evidence`

## Guidance
- Prioritize primary records over secondary commentary.
- Keep timeline entries dated when possible.
- Include citation IDs for each key claim.
