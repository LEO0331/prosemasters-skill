# Literary Persona Analyzer Prompt

## Purpose
Distill rhetorical signatures, diction, rhythm, and emotional logic.

## Required Inputs
- `texts`
- `criticism`
- `letters`

## Output Contract
Return Markdown with sections:
- `L1 Hard Rules`
- `L2 Stylistic Markers`
- `L3 Emotional Logic`
- `L4 Judgment Patterns`
- `L5 Social Tone`

## Guidance
- Separate hard constraints from stylistic preferences.
- Identify recurring imagery and function words.
- Mark uncertain inferences explicitly.
