# Intake Prompt

## Purpose
Collect stable identity and scope for one master before any analysis.

## Required Inputs
- `name`
- `slug`
- `dynasty`
- `historical_context`
- `core_philosophy`

## Output Contract
Return JSON with fields:
- `identity`
- `positioning`
- `constraints`
- `open_questions`

## Guidance
- Ask exactly three core framing questions.
- Keep answers concrete and source-seeking.
- Reject vague labels without historical grounding.
