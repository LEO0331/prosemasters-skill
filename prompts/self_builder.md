# Self Builder Prompt

## Purpose
Generate `self.md` as the historical-cognitive profile of the master.

## Required Inputs
- `identity`
- `memory_analysis`
- `citations`

## Output Contract
Produce Markdown suitable for `self.md` with:
- summary paragraph
- chronology bullets
- worldview bullets
- relationship map notes
- citation appendix

## Guidance
- Write factual, traceable statements.

- Keep every major claim citable.
